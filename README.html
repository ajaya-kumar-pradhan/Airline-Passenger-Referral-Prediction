01 — Project Overview
Why Referral Prediction Matters

In the fiercely competitive aviation industry, passenger satisfaction and word-of-mouth referrals are key growth levers. This project builds a binary classification system that predicts whether a passenger will recommend the airline to their network — enabling airlines to prioritize customer engagement, tailor loyalty programmes, and convert satisfied travellers into brand advocates.

02 — Methodology
End-to-End ML Pipeline
01
Data Collection & Loading
Loaded airline passenger reviews dataset from Excel. Dataset includes multi-airline reviews with ratings across 6 service dimensions and open-text customer reviews.
02
Data Cleaning & Preprocessing
Handled missing values, removed duplicate rows, parsed date columns, label-encoded categorical variables (traveller_type, cabin) using LabelEncoder.
03
Exploratory Data Analysis (EDA)
Visualised distributions, correlation heatmaps, class balance, and feature relationships using Matplotlib and Seaborn. Identified ground_service and cabin_service as top predictors.
04
Feature Engineering & NLP
Applied TF-IDF vectorisation on customer_review text, VIF analysis for multicollinearity, and created engineered interaction features from service ratings.
05
Model Building & Tuning
Trained Logistic Regression, SVM (RBF kernel), and Random Forest classifiers. Applied GridSearchCV for hyperparameter tuning. Used cross-validation for robust evaluation.
06
Model Evaluation & Explainability
Compared models on Accuracy, Recall, F1, and AUC-ROC. Applied ELI5 permutation importance to interpret feature contributions. Priority metric: Recall.
07
Deployment
Best model (SVM) serialised with Joblib. REST API built with FastAPI. Interactive frontend deployed on Streamlit with live user input and real-time prediction.

07 — Conclusion
Key Takeaways

Across Logistic Regression, Random Forest, and SVM, the SVM classifier emerged as the best performer — delivering the highest accuracy and recall. Ground service and cabin service were the strongest predictors of passenger advocacy. The deployed Streamlit application bridges the gap between a trained model and real-world business utility, enabling airline operations teams to act on predictive signals in real time.
